## Commands Table

### Task Commands

| Name       | Description                                      |
| ---------- | ------------------------------------------------ |
| h          | Display information about all supported commands |
| q          | Quit Ass-Igned                                   |
| c          | Chat with DeepSeek                               |
| add        | Add a task                                       |
| del        | Delete a task                                    |
| ls \[t/p]  | List all tasks \[sorted by time / priority]      |
| info \<id> | View complete information of a task              |
| done \<id> | Set a task to be done                            |
| ud \<id>   | Update a task (time, priority, title)            |
| *unnamed*  | A most urgent task                               |
